# Learning Record 0070 - GitHub Pull Request Template

## What changed

- Added `.github/PULL_REQUEST_TEMPLATE.md` as the default GitHub pull request checklist.
- The template asks contributors for summary, change type, verification evidence, course documentation checklist, and risk notes.
- Added Lesson 0079 and Reference 0079 to explain how PR templates complement CI and branch protection.

## Why it matters

CI and branch protection can block failing changes, but reviewers still need human context. The PR template makes contribution evidence explicit before review starts.
