# Learning Record 0079 - Code of Conduct Workflow

Lesson 0088 added a repository code of conduct for public learning collaboration.

Key artifacts:

- `CODE_OF_CONDUCT.md`
- `lessons/0088-code-of-conduct-workflow.html`
- `reference/0088-code-of-conduct-workflow-cheatsheet.html`

Core learning:

- A code of conduct is the collaboration safety layer around contribution workflow.
- Beginner-safe communities protect both politeness and learning quality.
- Maintainers need proportionate response options and evidence-preserving reporting rules.
- Public channels must not contain secrets, private logs, personal data, customer data, exploit details, or proprietary code.
