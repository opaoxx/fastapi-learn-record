# Learning Record 0081 - License Reuse Boundary

Lesson 0090 added the repository license and reuse-boundary explanation.

Key artifacts:

- `LICENSE`
- `lessons/0090-license-reuse-boundary.html`
- `reference/0090-license-reuse-boundary-cheatsheet.html`

Core learning:

- A public GitHub repository should state reuse rights explicitly.
- MIT License permits broad reuse, modification, distribution, sublicensing, and selling copies.
- Reusers must keep copyright and permission notices.
- MIT includes "AS IS", no-warranty, and no-liability language.
