# Learning Record 0077 - Changelog Maintenance Workflow

Lesson 0086 added a root changelog as the local release ledger for the course repository.

Key artifacts:

- `CHANGELOG.md`
- `lessons/0086-changelog-maintenance-workflow.html`
- `reference/0086-changelog-maintenance-workflow-cheatsheet.html`

Core learning:

- `CHANGELOG.md` records curated public changes.
- GitHub Release Notes summarize a published tag.
- `COURSE_PROGRESS.md` keeps detailed work-session evidence.
- Changelog entries should describe learner-visible and contributor-visible changes, not every internal editing step.
