# Learning Record 0078 - Contributing Guide Workflow

Lesson 0087 added the repository contribution guide.

Key artifacts:

- `CONTRIBUTING.md`
- `lessons/0087-contributing-guide-workflow.html`
- `reference/0087-contributing-guide-workflow-cheatsheet.html`

Core learning:

- A contribution guide is the contributor-facing control plane for issue, branch, PR, test, changelog, and review behavior.
- Lesson contributions must keep the fixed nine-section review structure.
- Pull requests must provide verification evidence, risk notes, and documentation checklist status.
- Contribution workflow changes belong in `CHANGELOG.md` under `Unreleased` until release.
