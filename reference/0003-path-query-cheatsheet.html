Reference 0003

路径参数和查询参数速查

路径参数定位一个资源，查询参数筛选一批资源。

路径参数

@app.get("/items/{item_id}")
def read_item(item_id: int):
    return {"item_id": item_id}

/items/2 中的 2 会进入 item_id。

查询参数

@app.get("/items")
def list_items(category: str | None = None):
    ...

/items?category=book 中的 book 会进入 category。

判断规则

  • 出现在路径模板里的函数参数是路径参数。
  • 没有出现在路径模板里的简单类型参数通常是查询参数。
  • Pydantic 模型参数通常来自请求体。

常见校验

item_id: Annotated[int, Path(ge=1)]
q: Annotated[str | None, Query(max_length=40)] = None
max_price: Annotated[float | None, Query(ge=0)] = None
来源：FastAPI Path Parameters、 FastAPI Query Parameters。
