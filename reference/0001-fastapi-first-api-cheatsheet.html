Reference 0001

FastAPI 第一个 API 速查

从“文件里有几行代码”到“浏览器能访问接口”的最小闭环。

最小结构

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello FastAPI"}

四个关键词

  • app: FastAPI 应用对象，所有接口都挂在它上面。
  • @app.get("/"): 路径操作装饰器，表示 GET 请求访问 /。
  • def read_root(): 路径操作函数，请求来了以后 FastAPI 调用它。
  • return {"message": ...}: 返回 Python 字典，FastAPI 会序列化成 JSON。

本地运行

python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install "fastapi[standard]"
fastapi dev first_api/main.py

常用地址

  • http://127.0.0.1:8000/: 根接口。
  • http://127.0.0.1:8000/health: 健康检查接口。
  • http://127.0.0.1:8000/docs: 自动生成的交互式 API 文档。
来源：FastAPI Tutorial、 FastAPI First Steps。
