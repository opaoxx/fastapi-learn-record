Reference 0068

Provider Metrics Exercise Grading Anchors 速查

用确定性锚点检查排障作答是否覆盖 severity 和 next_action。

核心函数

grade_provider_metrics_runbook_exercise_answer(card, response)

检查锚点

  • expected_severity：优先级判断。
  • expected_next_action：下一步动作。
  • casefold()：大小写归一化。

返回字段

  • card_id：追溯题卡。
  • severity_matched / next_action_matched：细粒度命中结果。
  • passed：两个锚点都命中。
  • missing：缺失字段列表。
  • feedback：人类可读反馈。

边界

  • 不做语义相似度评分。
  • 不调用 provider，不读取 counter。
  • 生产环境需要 rubric、复核、脱敏和审计。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
