Reference 0081

GitHub Issue Templates 速查

把 bug、课程需求和文档修复分流为结构化 GitHub Issue Forms。

核心目录

.github/ISSUE_TEMPLATE/
├── bug_report.yml
├── lesson_request.yml
├── documentation_fix.yml
└── config.yml

模板职责

  • bug_report.yml：复现步骤、期望行为、实际行为、验证命令。
  • lesson_request.yml：关联课程、学习缺口、所需深度、输出类型。
  • documentation_fix.yml：影响文件、文档问题、建议修复、课程标准影响。

Issue Forms 字段

name
description
title
labels
body[].type
body[].id
body[].attributes
body[].validations.required

协作链路

Issue template
-> labels and structured fields
-> maintainer triage
-> pull request
-> PR template
-> CI + CODEOWNERS + branch protection
