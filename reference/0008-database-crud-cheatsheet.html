Reference 0008

数据库 Session 和 CRUD 速查

Session 是一次请求访问数据库的工作单元。

Session 依赖

def get_session():
    with Session(engine) as session:
        yield session

查询列表

statement = select(Item)
items = list(session.exec(statement))

按主键查询

item = session.get(Item, item_id)

创建记录

session.add(item)
session.commit()
session.refresh(item)

关键词

  • add: 准备新增对象。
  • commit: 提交保存。
  • refresh: 从数据库刷新对象。
  • yield: 把资源交给路径函数，之后继续清理。
来源：FastAPI SQL Databases、 FastAPI Dependencies with yield。
