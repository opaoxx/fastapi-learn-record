Reference 0083

GitHub Projects Board Workflow 速查

把已 triage 的 issue 和 PR 投影成可排期、可跟踪、可复盘的 Projects 看板。

核心文件

.github/PROJECT_BOARD.md

必备字段

Status: Intake, Needs Repro, Needs Design, Ready, In Progress, In Review, Blocked, Done
Priority: P0, P1, P2, P3
Area: API, Tests, Lessons, Reference, GitHub
Type: Bug, Lesson, Docs, Workflow
Lesson: lesson number or range

推荐视图

Triage Queue
Ready Work
Course Docs
GitHub Workflow
Done Review

状态映射

status:needs-repro -> Needs Repro
status:needs-design -> Needs Design
status:ready -> Ready
status:blocked -> Blocked
status:done -> Done

核心边界

Projects 是 GitHub 远程配置；提交文档不会自动创建看板。CI、文档契约测试、CODEOWNERS 和 branch protection 仍是质量门禁。
