Reference 0024

任务状态筛选速查

用 status 查询参数只看某一种任务状态。

请求

GET /tasks?status=queued
GET /tasks?status=running
GET /tasks?status=completed
GET /tasks?status=failed

类型

TaskStatus = Literal["queued", "running", "completed", "failed"]

路由参数

task_status: Annotated[
    TaskStatus | None,
    Query(alias="status"),
] = None

数据库条件

if task_status is not None:
    statement = statement.where(SummaryTask.status == task_status)

组合

GET /tasks?status=failed&limit=10&offset=0

响应位置

response["items"]  # 当前页任务列表
response["count"]  # 当前筛选条件下的总数
来源：FastAPI Query Parameters、 SQLModel Limit and Offset。
