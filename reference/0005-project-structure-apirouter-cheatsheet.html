Reference 0005

项目结构和 APIRouter 速查

main.py 负责组装应用，router 文件负责声明接口。

推荐结构

first_api/
  main.py
  schemas.py
  database.py
  dependencies.py
  routers/
    system.py
    items.py
    predictions.py

创建 router

router = APIRouter(prefix="/items", tags=["items"])

@router.get("")
def list_items():
    ...

挂载 router

app.include_router(items.router)

职责划分

  • schemas.py: Pydantic 模型。
  • database.py: SQLite 设置、建表和数据库 Session。
  • dependencies.py: 可复用依赖。
  • routers/: 按业务领域拆分接口。
来源：FastAPI Bigger Applications。
