Reference 0012

API Key 鉴权速查

客户端发 X-API-Key，后端和 Settings 中的 key 比较。

Security 工具

api_key_header = APIKeyHeader(name="X-API-Key", auto_error=False)

校验依赖

def require_api_key(api_key, settings):
    if api_key != settings.api_key:
        raise HTTPException(status_code=401)

保护接口

@router.post("", dependencies=[Depends(require_api_key)])
def create_item(...):
    ...

请求示例

X-API-Key: dev-secret-key

状态码

  • 401: 缺少 key 或 key 错误。
  • 201: key 正确，创建成功。
  • 422: key 通过，但请求体不合法。
来源：FastAPI Security Tools、 FastAPI Security、 FastAPI Header Parameters。
