Reference 0021

AI 客户端错误速查

内部异常用 AIClientError，对外响应用 HTTPException。

自定义异常

raise AIClientError(
    message="The AI service is unavailable.",
    error_code="ai_client_unavailable",
)

错误响应

{
  "detail": {
    "error_code": "ai_client_unavailable",
    "message": "The AI service is unavailable."
  }
}

状态码选择

  • 422: 请求参数不合法。
  • 401: 鉴权失败。
  • 404: 资源不存在。
  • 503: 模型或外部服务暂时不可用。
来源：FastAPI Handling Errors、 MDN 503 Service Unavailable。
