Reference 0020

AI 客户端依赖速查

把模型调用封装成客户端，再用 Depends 交给路由。

客户端

class DemoAIClient:
    def predict_sentiment(self, text: str, mode: str) -> PredictionResult:
        ...

    def summarize(self, text: str) -> str:
        ...

依赖入口

def get_ai_client() -> DemoAIClient:
    return DemoAIClient()

路由使用

def predict(
    payload: PredictionRequest,
    ai_client: Annotated[DemoAIClient, Depends(get_ai_client)],
) -> PredictionResponse:
    prediction = ai_client.predict_sentiment(payload.text, payload.mode)

测试替换

app.dependency_overrides[get_ai_client] = override_ai_client

关键区别

  • AI client：内部模型调用能力。
  • Response model：对外 JSON 合约。
  • dependency override：测试时替换依赖。
来源：FastAPI Dependencies、 FastAPI Testing Dependencies with Overrides、 Python dataclasses。
