Reference 0007

SQLite 和 SQLModel 速查

SQLite 保存数据，SQLModel 把 Python 类映射成数据库表。

表模型

class Item(ItemBase, table=True):
    id: int | None = Field(default=None, primary_key=True)

数据库引擎

DATABASE_URL = "sqlite:///first_api.db"
engine = create_engine(DATABASE_URL)

建表

SQLModel.metadata.create_all(engine)

模型分层

  • ItemBase: 公共字段。
  • Item: 数据库表模型。
  • ItemCreate: 创建请求模型。
  • ItemRead: 响应读取模型。
来源：FastAPI SQL Databases、 SQLModel Simple Hero API。
