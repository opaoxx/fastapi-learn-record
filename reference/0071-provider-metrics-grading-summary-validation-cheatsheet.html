Reference 0071

Provider Metrics Grading Summary Validation 速查

渲染 Markdown 报告前，先检查 summary 的结构和统计一致性。

核心函数

validate_provider_metrics_runbook_grading_summary(summary)

校验重点

  • total/answered/passed/failed/unanswered 必须存在且是整数。
  • grades 必须是 list。
  • 每个 grade 必须有 card_id、answered、passed、missing、feedback。
  • failed == total - passed。
  • unanswered == total - answered。
  • total == len(grades)。

返回结构

  • valid：是否通过校验。
  • errors：完整错误列表。

边界

  • 不修复坏数据。
  • 不重新评分。
  • 不渲染 Markdown。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
