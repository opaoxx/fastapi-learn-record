Reference 0017

CORS 速查

CORS 是浏览器跨源访问 API 时的许可机制。

Origin

origin = protocol + host + port
  • http://127.0.0.1:8000 和 http://127.0.0.1:5500 不同源。
  • http://localhost:8000 和 http://127.0.0.1:8000 不同源。

FastAPI 配置

from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://127.0.0.1:5500"],
    allow_methods=["*"],
    allow_headers=["*"],
)

预检请求

OPTIONS /files/text
Origin: http://127.0.0.1:5500
Access-Control-Request-Method: POST
Access-Control-Request-Headers: X-API-Key

常见误区

  • CORS 错误主要发生在浏览器里，不是所有 HTTP 客户端都会遇到。
  • 端口不同就是不同源。
  • 自定义请求头通常会触发预检请求。
  • 真实项目不要随意允许所有来源。
来源：FastAPI CORS、 FastAPI Middleware。
