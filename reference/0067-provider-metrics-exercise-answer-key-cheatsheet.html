Reference 0067

Provider Metrics Exercise Answer Key 速查

把题卡渲染成标准答案键，完成 runbook 训练闭环。

核心函数

render_provider_metrics_runbook_exercise_answer_key(cards)

答案块结构

  • ## card id
  • Prompt：保留题目上下文。
  • Expected severity：判断锚点。
  • Answer：标准解释和动作。

边界

  • 只渲染答案，不评分用户输入。
  • 从 cards 自动生成，不手写散落答案。
  • 适合 GitHub 离线复习和面试训练。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
