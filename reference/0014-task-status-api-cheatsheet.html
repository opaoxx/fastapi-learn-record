Reference 0014

任务状态 API 速查

提交任务拿 ID，查询任务看状态和结果。

状态流转

queued -> running -> completed
queued -> running -> failed

表模型

class SummaryTask(SQLModel, table=True):
    id: int | None = Field(default=None, primary_key=True)
    text: str
    text_length: int
    status: str = "queued"
    result: str | None = None
    error: str | None = None

接口

  • POST /tasks/summaries: 提交摘要任务，返回 202。
  • GET /tasks/{task_id}: 查询单个任务。
  • GET /tasks: 查询任务列表。

AI 服务映射

  • 输入文本 -> prompt 或用户内容。
  • 后台函数 -> 模型调用。
  • result -> 模型输出。
  • status -> 前端进度展示。
来源：FastAPI Background Tasks、 MDN 202 Accepted。
