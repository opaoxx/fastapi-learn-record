Reference 0011

配置和环境变量速查

配置放 Settings，秘密放环境变量，示例放 .env.example。

Settings 类

class Settings(BaseSettings):
    app_name: str = "First FastAPI API"
    environment: str = "development"
    api_key: str = "dev-secret-key"

环境变量前缀

FIRST_API_APP_NAME
FIRST_API_ENVIRONMENT
FIRST_API_API_KEY

缓存

@lru_cache
def get_settings() -> Settings:
    return Settings()

规则

  • .env.example 可以提交。
  • .env 不提交。
  • 公开接口不要返回真实密钥。
  • 测试或部署时可以用环境变量覆盖默认值。
来源：FastAPI Settings、 Pydantic Settings。
