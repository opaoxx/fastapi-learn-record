Reference 0072

Provider Metrics Practice Session Package 速查

把 runbook 训练从样本到报告打包成完整会话对象。

核心函数

build_provider_metrics_runbook_practice_session(responses_by_card_id)

输出字段

  • samples：固定指标练习样本。
  • findings：排障解释和动作。
  • findings_markdown：findings 表格报告。
  • cards：训练题卡。
  • answer_key：标准答案键。
  • summary：评分汇总。
  • validation：summary 校验结果。
  • report_markdown：评分复盘报告。

关键原则

  • 编排层不复制评分规则。
  • validation 通过后才渲染报告。
  • 不读取 provider、counter 或数据库。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
