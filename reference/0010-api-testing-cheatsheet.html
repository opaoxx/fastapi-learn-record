Reference 0010

API 自动化测试速查

TestClient 发请求，pytest 跑测试，dependency_overrides 替换依赖。

最小测试

with TestClient(app) as client:
    response = client.get("/health")
    assert response.status_code == 200

依赖覆盖

app.dependency_overrides[get_session] = override_get_session

运行测试

python -m pytest -q

常测内容

  • 成功路径：200、201、204。
  • 业务错误：404。
  • 校验错误：422。
  • 响应 JSON 的关键字段。
来源：FastAPI Testing、 FastAPI Testing Dependencies with Overrides。
