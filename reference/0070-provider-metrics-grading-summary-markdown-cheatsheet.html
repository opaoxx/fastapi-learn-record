Reference 0070

Provider Metrics Grading Summary Markdown 速查

把结构化评分汇总渲染成 GitHub 友好的 Markdown 复盘报告。

核心函数

render_provider_metrics_runbook_grading_summary_markdown(summary)

报告结构

  • Provider Metrics Runbook Grading Summary：总体统计。
  • Grade Details：逐题明细。
  • missing 为空时显示 -。

输入契约

  • summary["grades"] 必须是 list。
  • grade["missing"] 必须是 list。
  • renderer 不重新计算评分字段。

边界

  • 不展示原始 response。
  • 不读取 provider 或 counter。
  • 生产环境应增加 Markdown 转义和权限治理。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
