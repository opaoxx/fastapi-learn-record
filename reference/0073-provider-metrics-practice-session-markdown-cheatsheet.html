Reference 0073

Provider Metrics Practice Session Markdown 速查

把完整 practice session 渲染成 GitHub 友好的 Markdown 复盘包。

核心函数

render_provider_metrics_runbook_practice_session_markdown(session)

章节顺序

  • Findings：指标解释表。
  • Exercise Cards：题卡表。
  • Answer Key：标准答案。
  • Grading Report：评分报告。

边界

  • 不重新 build session。
  • 不重新评分。
  • 题卡表不展示完整 next_action。
  • cards 必须是 list。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
