Reference 0069

Provider Metrics Grading Summary 速查

把多题评分聚合成可复盘的训练汇总。

核心函数

summarize_provider_metrics_runbook_exercise_grades(cards, responses_by_card_id)

输入结构

  • cards：题卡列表。
  • responses_by_card_id：以 card["id"] 为 key 的作答字典。

输出字段

  • total：题目总数。
  • answered：已作答数。
  • passed：通过数。
  • failed：未通过数。
  • unanswered：未作答数。
  • grades：逐题评分结果。

关键原则

  • 未作答也生成 grade，避免统计分母失真。
  • summary 不保存 response 原文。
  • 批量层复用单题评分函数，不复制规则。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
