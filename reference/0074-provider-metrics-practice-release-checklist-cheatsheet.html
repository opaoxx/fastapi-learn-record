Reference 0074

Provider Metrics Practice Release Checklist 速查

GitHub 发布完整训练包前，检查关键材料是否齐全。

核心函数

build_provider_metrics_runbook_practice_release_checklist(session, package_markdown)

检查项

  • findings_markdown_present
  • exercise_cards_present
  • answer_key_present
  • grading_summary_valid
  • grading_report_present
  • package_sections_present

返回结构

  • ready：全部检查通过才为 true。
  • checks：逐项 id/passed/evidence。

边界

  • 不生成缺失材料。
  • 不重新评分。
  • 不替代生产 CI。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
