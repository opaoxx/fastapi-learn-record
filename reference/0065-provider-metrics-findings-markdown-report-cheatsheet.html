Reference 0065

Provider Metrics Findings Markdown Report 速查

把 findings 渲染成可复盘、可复制、可测试的 Markdown 表格。

核心函数

render_provider_metrics_runbook_findings_markdown(findings)

输出列

  • outcome：排障分支。
  • count：样本数量，右对齐。
  • severity：教学级判断。
  • finding：人类解释。
  • next_action：下一步动作。

边界

  • 只渲染文本，不读取 counter。
  • 不把 finding 写回 metric label。
  • 完整字符串测试保护表格格式。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
