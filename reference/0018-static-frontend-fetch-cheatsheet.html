Reference 0018

静态前端调用 API 速查

页面负责收集输入，fetch 负责调用 API，后端负责处理和返回 JSON。

挂载静态页

app.mount("/app", StaticFiles(directory=frontend_dir, html=True), name="frontend")

上传文件

const formData = new FormData();
formData.append("file", file);

await fetch("/files/text", {
  method: "POST",
  headers: {"X-API-Key": apiKey},
  body: formData,
});

调用任务

POST /tasks/files/{file_id}/summary
GET /tasks/{task_id}

关键提醒

  • FormData 上传文件时不要手写 Content-Type。
  • fetch 遇到 4xx/5xx 不会自动抛异常。
  • 同源页面不触发 CORS，独立前端开发服务器通常会触发 CORS。
  • 浏览器页面调用受保护接口时，要带上 X-API-Key。
来源：FastAPI Static Files、 FastAPI CORS。
