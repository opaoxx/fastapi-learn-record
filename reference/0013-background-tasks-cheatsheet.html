Reference 0013

BackgroundTasks 速查

先返回响应，再执行轻量后台工作。

路径函数参数

def create_task(background_tasks: BackgroundTasks):
    background_tasks.add_task(worker, task_id)
    return task

适合场景

  • 发送邮件。
  • 写日志或审计记录。
  • 轻量数据处理。
  • 入门级 AI 任务 demo。

限制

  • 和 FastAPI 应用在同一进程。
  • 服务重启可能丢失运行中任务。
  • 没有内置重试和分布式 worker。
来源：FastAPI Background Tasks。
