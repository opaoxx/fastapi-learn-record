Reference 0055

Provider Metrics Export Shape 速查

同一份 provider metrics 可以用 JSON snapshot 调试，也可以渲染成 Prometheus text 风格输出供采集器读取。

三层模型

ProviderMetricsCounter
  -> JSON snapshot: [{"labels": ..., "count": 1}]
  -> text export: provider_prediction_total{...} 1

Endpoint

GET /provider/metrics/prometheus

返回 text/plain，不是 JSON。

文本输出

# HELP provider_prediction_total Provider prediction outcomes recorded by the teaching metrics counter.
# TYPE provider_prediction_total counter
provider_prediction_total{error_code="none",failure_category="none",operation="prediction",outcome="success",status_code="none"} 1

核心 helper

def render_provider_metrics_prometheus_text(samples: list[dict[str, object]]) -> str:
    lines = [
        f"# HELP {PROVIDER_PREDICTION_METRIC_NAME} {PROVIDER_PREDICTION_METRIC_HELP}",
        f"# TYPE {PROVIDER_PREDICTION_METRIC_NAME} counter",
    ]
    for sample in samples:
        labels = sample["labels"]
        label_text = ",".join(
            f'{name}="{escape_prometheus_label_value(value)}"'
            for name, value in sorted(labels.items())
        )
        lines.append(f"{PROVIDER_PREDICTION_METRIC_NAME}{{{label_text}}} {sample['count']}")
    return "\n".join(lines) + "\n"

FastAPI router

@router.get(
    "/provider/metrics/prometheus",
    response_class=PlainTextResponse,
)
async def get_provider_metrics_prometheus() -> str:
    return get_provider_metrics_prometheus_text()

格式规则

  • # HELP 描述指标含义。
  • # TYPE 描述指标类型。
  • provider_prediction_total 是 counter 名称。
  • labels 用稳定顺序输出。
  • label value 里的反斜杠、换行和双引号要转义。

测试重点

response = client.get("/provider/metrics/prometheus")

assert response.status_code == 200
assert response.headers["content-type"].startswith("text/plain")
assert response.text == expected_text

常见错误

  • 把教学 text endpoint 当生产监控系统。
  • label 顺序不稳定。
  • 没有转义 label value。
  • 对文本 endpoint 套 JSON response model。
  • 随意修改已经暴露的 metric name。

面试短答

metrics 的事实和导出形状要分开。内部 counter 负责累计，JSON snapshot 适合调试，Prometheus text 风格输出适合展示采集器读取的行式样本。即使是教学实现，也要保证 metric name、label 顺序和转义规则稳定。
