Reference 0006

Depends 依赖注入速查

路径函数声明需要什么，FastAPI 自动准备并注入。

类依赖

class Pagination:
    def __init__(self, skip: int = 0, limit: int = 10):
        self.skip = skip
        self.limit = limit

使用 Depends

def list_items(
    pagination: Annotated[Pagination, Depends(Pagination)],
):
    return results[pagination.skip : pagination.skip + pagination.limit]

适合抽成依赖的内容

  • 分页、排序、筛选等公共查询参数。
  • 数据库会话。
  • 当前用户和权限检查。
  • AI 模型客户端、向量数据库客户端。
  • 请求追踪信息和日志上下文。

执行顺序

请求进入 -> 执行依赖 -> 注入参数 -> 执行路径函数
来源：FastAPI Dependencies、 FastAPI Classes as Dependencies。
