Reference 0025

列表响应信封速查

列表数据放 items，分页元数据放外层字段。

响应模型

class SummaryTaskListResponse(BaseModel):
    items: list[SummaryTaskRead]
    count: int
    limit: int
    offset: int

响应示例

{
  "items": [],
  "count": 0,
  "limit": 20,
  "offset": 0
}

字段含义

  • items: 当前页数据。
  • count: 当前筛选条件下的总数。
  • limit: 当前页最多返回多少条。
  • offset: 当前页跳过了多少条。
来源：FastAPI Response Model、 FastAPI Query Parameters。
