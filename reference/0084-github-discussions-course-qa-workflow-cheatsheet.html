Reference 0084

GitHub Discussions Course Q&A Workflow 速查

把学习问答、笔记、想法、展示和公告从 issue 队列中分离出来。

核心文件

.github/DISCUSSIONS.md

推荐分类

Q&A
Study Notes
Ideas
Show and Tell
Announcements

分流规则

Issue:
reproducible + scoped + repository files should change

Discussion:
exploratory + open-ended + primarily educational

转 Issue 流程

discussion question
-> answer with lesson/source/test links
-> identify repository work
-> open issue with template
-> link discussion and issue
-> triage labels
-> project board
-> pull request
