Reference 0002 Revised

请求体和 Pydantic 速查

一句话：Pydantic 模型是 FastAPI 接口边界上的数据协议。

请求体是什么

请求体是客户端随 HTTP 请求发送给 API 的数据。AI 服务常用 POST 请求体传文本、图片地址、模型参数。

Pydantic 模型负责什么

  • 定义 JSON 应该有哪些字段。
  • 定义字段类型，例如 str、float。
  • 定义字段约束，例如长度、范围、枚举值。
  • 把合法输入转换成 Python 对象。
  • 把模型结构交给 FastAPI 生成 /docs 文档。

本课模型

class PredictionRequest(BaseModel):
    text: str = Field(min_length=3, max_length=500)
    mode: Literal["fast", "careful"] = "fast"
    temperature: float = Field(default=0.2, ge=0, le=1)

字段含义

  • text: 核心输入文本，必填，长度 3 到 500。
  • mode="fast": 默认快速模式，适合低成本、快响应。
  • mode="careful": 谨慎模式，适合更稳、更慢、更贵的处理策略。
  • temperature: AI 参数雏形，范围 0 到 1，默认 0.2。

默认值不是可选值

Literal["fast", "careful"] = "fast" 只允许两个值："fast" 和 "careful"。后面的 = "fast" 表示不传时自动使用 "fast"。

422 怎么读

422 表示请求 JSON 能被服务器理解，但内容不符合接口模型。常见原因：

  • 缺少必填字段。
  • 字段类型错误。
  • 枚举值不合法。
  • 长度或数值范围不合法。
