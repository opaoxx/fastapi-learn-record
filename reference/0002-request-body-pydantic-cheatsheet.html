请求体和 Pydantic 速查

写 AI 服务接口时，最常见的形状就是：客户端发 JSON，请求模型校验，函数处理，响应模型返回。

最小 POST 请求体

from pydantic import BaseModel, Field

class PredictionRequest(BaseModel):
    text: str = Field(min_length=3, max_length=500)

@app.post("/predict")
def predict(payload: PredictionRequest):
    return {"text_length": len(payload.text)}

字段写法

  • text: str: 必填字符串字段。
  • mode: Literal["fast", "careful"]: 只能取固定选项。
  • mode = "fast": 有默认值，所以请求里可以不传。
  • Field(min_length=3): 给字段加验证规则和文档元数据。

有效 JSON 示例

{
  "text": "FastAPI is pleasant",
  "mode": "careful"
}

常见 422

  • 缺少必填字段：没有传 text。
  • 字段类型错误：text 传了数字或对象。
  • 字段值不合法：mode 不是 fast 或 careful。
  • 长度不合法：text 太短或太长。
来源：FastAPI Request Body、 FastAPI Body Fields。
