Reference 0016

文件摘要任务速查

上传文件拿 file_id，创建任务拿 task_id，查询任务拿 result。

工作流

POST /files/text
POST /tasks/files/{file_id}/summary
GET /tasks/{task_id}

任务来源

source_file_id=uploaded_file.id

状态码

  • 201: 文件上传成功。
  • 202: 文件摘要任务已接收。
  • 401: 缺少或错误 API Key。
  • 404: 文件或任务不存在。
  • 422: 文件类型、编码或请求参数不合法。
来源：FastAPI Request Files、 FastAPI Background Tasks。
