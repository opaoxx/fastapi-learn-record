Reference 0019

服务层速查

router 管 HTTP，service 管业务，model 管数据形状。

目录

first_api/services/
  summary_tasks.py
  ai_clients.py

判断规则

  • HTTP 参数、状态码、请求头：留在 router。
  • 创建任务、运行任务、组合业务步骤：放进 service。
  • 字段、表、校验约束：放进 schema/model。
  • 多个地方要复用：优先考虑 service。

典型服务函数

def create_summary_task(session: Session, text: str) -> SummaryTask:
    task = SummaryTask(text=text, text_length=len(text), status="queued")
    session.add(task)
    session.commit()
    session.refresh(task)
    return task
来源：FastAPI Bigger Applications、 FastAPI Dependencies。
