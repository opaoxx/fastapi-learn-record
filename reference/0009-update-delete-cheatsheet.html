Reference 0009

更新和删除速查

PATCH 做部分更新，DELETE 删除资源。

部分更新模型

class ItemUpdate(SQLModel):
    name: str | None = None
    category: ItemCategory | None = None
    price: float | None = Field(default=None, ge=0)
    in_stock: bool | None = None

更新核心

update_data = payload.model_dump(exclude_unset=True)
item.sqlmodel_update(update_data)
session.add(item)
session.commit()
session.refresh(item)

删除核心

session.delete(item)
session.commit()

状态码

  • PATCH 成功：200，返回更新后的对象。
  • DELETE 成功：204，无响应体。
  • 资源不存在：404。
  • 更新内容非法：422。
来源：SQLModel Update Data with FastAPI、 SQLModel Delete Data with FastAPI。
