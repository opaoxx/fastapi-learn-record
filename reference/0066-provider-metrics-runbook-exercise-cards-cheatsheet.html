Reference 0066

Provider Metrics Runbook Exercise Cards 速查

把 findings 变成可自测的 runbook 题卡。

核心函数

build_provider_metrics_runbook_exercise_cards(findings)

题卡字段

  • id：稳定题号。
  • prompt：只暴露 outcome/count。
  • expected_severity：优先级锚点。
  • expected_next_action：评分动作锚点。
  • answer：finding + next_action。

关键原则

  • prompt 不泄露答案。
  • 题卡从 findings 生成，不手写漂移。
  • 稳定 id 便于文档引用。

测试命令

.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
