Reference 0095

Release Candidate Versioned Publishing Readiness 速查

用 .github/RELEASE_CANDIDATE.md 管理课程发布候选、冻结、验证证据和 tag 就绪。

RC 进入条件

lesson range explicit
lesson/reference/index aligned
COURSE_PROGRESS evidence
CHANGELOG Unreleased accurate
FINAL_AUDIT reviewed
LINK_AUDIT reviewed

冻结规则

no new lesson topics
no filename rename without link updates
no public API change without tests and lessons
no real secrets
small traceable fixes only

验证命令

.\.venv\Scripts\python.exe -m pytest tests\test_course_docs_contract.py -q
.\.venv\Scripts\python.exe -m pytest tests\test_provider_http.py -q
.\.venv\Scripts\python.exe -m pytest -q
git diff --check

发布就绪

tag: course-vYYYY.MM.DD-N
use annotated tag
prepare Release Notes
record known warnings
link rollback rules

退出条件

all checks pass
known warnings documented
release range explicit
no broken links
no missing references
no secret leaks
