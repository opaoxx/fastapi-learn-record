Reference 0015

UploadFile 速查

文件上传来自 multipart/form-data，用 UploadFile 接收。

接口参数

async def upload_text_file(
    file: Annotated[UploadFile, File()],
):
    raw_content = await file.read()

常用属性

  • file.filename: 上传文件名。
  • file.content_type: 内容类型。
  • await file.read(): 读取文件字节。

校验

  • 扩展名：只收 .txt。
  • 大小：限制最大字节数。
  • 编码：要求 UTF-8。
  • 鉴权：上传接口需要 X-API-Key。
来源：FastAPI Request Files、 FastAPI UploadFile Reference。
