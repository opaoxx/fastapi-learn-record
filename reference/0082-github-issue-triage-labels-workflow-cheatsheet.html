Reference 0082

GitHub Issue Triage Labels Workflow 速查

用 type、area、priority、status 把 issue 队列治理成可筛选任务流。

核心文件

.github/ISSUE_TRIAGE.md

标签维度

type:bug | type:lesson | type:docs | type:workflow
area:api | area:tests | area:lessons | area:reference | area:github
priority:p0 | priority:p1 | priority:p2 | priority:p3
status:needs-repro | status:needs-design | status:ready | status:blocked | status:done

状态流

new issue
-> status:needs-repro
-> status:needs-design
-> status:ready
-> pull request opened
-> status:blocked
-> status:done

Triage 不变量

  • Exactly one type label.
  • At least one area label.
  • Exactly one priority label.
  • Exactly one status label.
